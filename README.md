# Sobre

O Modulo Gubee Integration é a integração entre a API da Gubee e o Magento 2, com o objetivo de disponibilizar os produtos da Gubee para venda no Magento 2.

# Instalação

Para instalar o módulo, siga os passos abaixo, na raiz do projeto Magento 2:
```bash
composer require gubee/integration
```


# Documentação

A documentação completa da biblioteca pode ser encontrada em [https://apidocs.gubee.com.br](https://apidocs.gubee.com.br). Swagger: [api.gubee.com.br/api/swagger-ui/](https://api.gubee.com.br/api/swagger-ui/#/)
